About

This game was written during LDJam 56. In this community driven game competition you build a game from scratch in 48 hours. Which means you design the concept, program behaviour and mechanics, make art and audio and play test the game in 48 hours.

This game was made between 2024-10-5 to 2024-10-6 CEST

Tools used:

  • Visual Studio IDE
  • Audacity
  • Anvil
  • Krita
  • Adobe Illustrator
  • Microsoft Edge
